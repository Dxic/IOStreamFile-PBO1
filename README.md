## Deskripsi
Project ini berisi program Java sederhana untuk mengelola data barang toko menggunakan konsep OOP, ArrayList, dan file text.

## Struktur Package
File diletakkan pada package `tugas`:

- `Barang.java` sebagai class model barang.
- `Gudang.java` sebagai class pengelola data barang.
- `MainTugas.java` sebagai class utama untuk menjalankan program.

## Fitur
- Menampilkan daftar kategori barang dari array String.
- Menambah minimal 5 data barang.
- Menampilkan semua barang.
- Menyimpan data barang ke file `barang.txt`.
- Memuat kembali data dari file ke objek Gudang baru.
- Menghitung total nilai persediaan dari harga dikali stok.

## Cara Menjalankan di NetBeans
1. Buat project Java Application bernama `IOStreamFile`.
2. Buat package bernama `tugas`.
3. Masukkan tiga file `.java` ke package `tugas`.
4. Jalankan file `MainTugas.java` dengan klik kanan > Run File atau tekan Shift+F6.
